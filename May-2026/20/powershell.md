Get-Process : finding process information

Get-Service : find information about windows services

Get-HotFix : identifying installed windows hot fixes

Get-NetAdapter : find information about network adapters

Get-NetConnectionProfile : get the network connection profile

Get-Culture : get the current culture settings. This includes information about the keyboard layout,
and the display format of items such as numbers, currency, and dates.

Get-UICulture : current user interface settings for windows. The UI culture settings determine which text strings appear in user interface elements such as menus and error messages.
